FAQ

Is coCoder Free?

Yes! This project is both open source and free! You can use it as much as you want, but we'll be also happy to accept donations - in case you want to give us one. Such contributions will make it much simpler for us to develop this amazing tool even further.

How long can I spend in a single session?

As long as you want! The session will be alive as long as any user is active inside of it.

Are old sessions garbage collected?

Yes! Every session that was inactive for 7 days (i.e. no user had it open in their browser) will be garbage collected.

Is there any limit for a number of users within single session?

There's no such limit at the moment. Feel free to use coCoder for one to many sessions as well!

How big can be the code chunk stored within a single session?

At this time we set the limit to 5MB. In case that's not enough for you, please feel free to open an issue for that.
